### Available Scripts
In the project directory, you can run:  

* Start json-server  

Run a fake server and test the app
```
yarn json-server --watch db.json --port 4000
```

* Run React  

Runs the app in the development mode.
Open http://localhost:3000 to view it in your browser.
```
yarn start
```
